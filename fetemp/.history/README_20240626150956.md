# Farm + Vue

This template should help you start developing using Vue and TypeScript in Farm.

## Setup

Install the dependencies:

```bash
# 首次安装farm
npm install -g farm-cli

# pnpm 安装依赖
pnpm install
```

## Get Started

- 启动本地项目:

```bash
pnpm start
```

- 打包本地项目:

```bash
pnpm build
```

- 预览打包后的本地项目:

```bash
pnpm preview
```

- 清除持久缓存本地文件

```bash
pnpm clean
```
